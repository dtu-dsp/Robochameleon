Detailed Description

IQ modulator model.

Takes 5 signal_interfaces (4 with data and a laser) and puts it in to one with 2 complex columns plus phase noise and carrier frequency of the laser If 'mode' is set to 'complex', it takes 3 signals (2 with the complex data and the laser) If 'mode' is set to 'single', it takes 2 signals (1 with 2 complex colums and the laser)
