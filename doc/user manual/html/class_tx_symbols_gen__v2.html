Detailed Description

This class implements the mapping between PRBS sequences and constellation symbols for polarization multiplexed coherent transmission.

Modulation Formats available:

  • 1.PDM-QPSK.
  • 2.PDM-16QAM.
  • 3.PDM-64QAM.
  • 4.Time Domain Hybrid QPSK\16QAM (Two options of multiplexing).
  • 5.Polarization Domain Hybrid QPSK\16QAM.

SymbolGenerator = TxSymbolsGen_v2(param)

Set of parameters required to create: 

*   param.NumberOfSymbols;
*   param.ModulationFormat;
*   param.SymbolRate;
*   param.PRBS_order;
*   param.DataFolder;
*   param.DataName;  
*

Traverse syntax : Eout = traverse(obj)
