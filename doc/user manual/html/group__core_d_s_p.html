Detailed Description

Stable, standard, DSP blocks.

DSP blocks which are both stable (working in most cases) and also standard (meant for use as a benchmark, in non-DSP-related experiments, etc.).

To add a class to this group, add the line:

@ingroup coreDSP

to the comment header.
